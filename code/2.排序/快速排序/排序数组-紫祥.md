## 排序数组

给定一个整数数组 nums，将该数组升序排列。


示例1:

> 输入: [5,2,3,1]
>
> 输出: [1,2,3,5]

示例2:

> 输入: [5,1,1,2,0,0]
>
> 输出: [0,0,1,1,2,5]

#### 提示：

1. 1<= A.length <= 10000
2. -50000 <= A[i] <= 50000

## 题解分析

采用sort()方法实现快速排序

sort()方法的原理是，当元素小于十个时候实现插入排序，当元素大于十个时候实现快速排序

## 代码片段

```JS
var sortArray = function(nums) {
    return nums.sort((a,b) => {
        return a-b;
    });
};
```

## 复杂度分析

时间复杂度：O(n log n)
空间复杂度：O(1)
